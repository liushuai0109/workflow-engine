# User Lifecycle Operations Management Platform

A comprehensive platform for managing user lifecycle operations through the AARRR framework.

## Monorepo Structure

- **@lifecycle/client** - Vue 3 frontend with BPMN editor  
- **@lifecycle/server** - Node.js Express backend API

## Quick Start

```bash
npm install
npm run bootstrap
npm run dev
```

See packages/client and packages/server for details.
