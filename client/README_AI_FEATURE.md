## 🎯 关键词触发

包含以下词语会启动 AI 流程图模式：
- 流程、流程图、画
- 创建、生成、添加、修改、删除
- 节点、任务、网关
- 开始、结束

## 💡 技术原理

AI 使用 **Function Calling** 技术，直接调用编辑器 API：

```
用户输入 → AI 理解需求 → 调用编辑器工具 → 实时绘制
```

可调用的工具：
- `createNode()` - 创建节点
- `createFlow()` - 创建连线
- `deleteNode()` - 删除节点
- `updateNode()` - 更新节点
- `clearCanvas()` - 清空画布
- `getNodes()` - 查看节点

## 🆚 与传统方式的区别

**传统方式**：AI 生成 XML 代码 → 解析 → 渲染
**新方式**：AI 直接操作编辑器 → 实时绘制

用户体验完全不同！现在您可以看到 AI "亲手"画图的过程。
