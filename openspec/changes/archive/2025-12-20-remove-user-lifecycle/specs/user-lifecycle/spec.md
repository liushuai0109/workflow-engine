# user-lifecycle 规范增量

## REMOVED Requirements

- AARRR 生命周期阶段定义
- 用户细分管理
- 用户旅程映射
- 生命周期指标跟踪
- 生命周期阶段过渡

**原因**：用户生命周期管理功能不再符合项目的核心目标。该功能属于业务运营和营销自动化领域，超出了 BPMN/XPMN 图表编辑器的产品定位范围。

**迁移**：
- 依赖此功能的用户应迁移至专门的营销自动化或用户运营平台（如 Mixpanel、Amplitude、Customer.io 等）
- 现有工作流可以导出为 BPMN/XPMN 文件保存
- 可以使用专业的客户数据平台（CDP）或分析工具来管理用户细分和旅程
