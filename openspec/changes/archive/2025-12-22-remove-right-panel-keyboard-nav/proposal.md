# 变更：删除右侧面板的键盘左右导航功能

## Why

右侧面板的键盘左右箭头导航功能会与编辑器其他交互产生冲突，并且可能导致意外的 Tab 切换。用户通常依赖鼠标点击或触摸来切换 Tab，键盘导航功能的使用频率极低，但却增加了复杂性和潜在的 UX 问题。

移除此功能可以：
1. 简化右侧面板组件的代码逻辑
2. 避免键盘事件冲突
3. 提供更可预测的用户体验
4. 减少不必要的交互模式

## What Changes

- **REMOVED**: 右侧面板 Tab 的键盘左右箭头导航功能
- 移除 `@keydown` 事件监听器
- 移除 `handleKeyDown` 函数和相关逻辑
- 移除 `tabKeys` 数组定义
- 移除容器的 `tabindex="0"` 属性
- 移除相关的焦点样式（`:focus-visible` 伪类）

## Impact

- **受影响的规范**: `workflow-editor` (右侧面板交互行为)
- **受影响的代码**:
  - `client/src/components/RightPanelContainer.vue` (lines 2, 116-137, 223-229)
- **破坏性变更**: 否（移除的是低频使用的辅助功能）
- **用户影响**: 极小，用户仍可通过点击 Tab 标签进行切换
